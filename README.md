# voltsitud-pangakoned

Fake bank calls fraud detection — defense project for Intro to Data Science (Tartu Ülikool Narva Kolledž).

## Repo layout

```
dataset/      # raw data ops: generation, dedup, QA, timestamp utils
analysis/     # cleaning + EDA + per-question visualizations (Sprint 2)
models/       # ML training / tuning / persistence (Sprint 3-5)
dashboard/    # Streamlit dashboard (Sprint 6)
docs/         # markdown documentation
  figures/    # committed PNG outputs from analysis/
data/         # CSVs (gitignored — regenerated by scripts)
```

## How to run (current state)

```bash
pip install -r requirements.txt

# Sprint 1 — clean dataset
python -m dataset.generate                    # produces data/raw_bank_calls.csv
python -m dataset.dedup                       # raw → data/dedup_bank_calls.csv + docs/dedup_report.md
python analysis/cleaning.py                   # raw → data/cleaned_bank_calls.csv (categoricals normalized)
python dataset/qa.py                          # 6 hard checks + statistics on the raw file

# Sprint 2 — EDA visualisations (each writes a PNG under docs/figures/)
python analysis/call_frequency.py
python -m analysis.suspicious_callers
python -m analysis.complaint_keywords
python analysis/subsequent_events.py
python analysis/temporal_features.py

# Sprint 3 — model-ready train/test data
python -m models.targets                      # labeled_bank_calls.csv (drops ambiguous labels)
python -m models.prepare                      # train/test (unscaled + scaled) + scaler.joblib
```
